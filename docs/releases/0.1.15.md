# Release 0.1.15

## Summary

`ccw-mcp` `0.1.15` implements deterministic ranking topicality guardrails for
the downstream lane-quality audit. It keeps the compiler deterministic and
generic: no LLM reranker, vector store, semantic-search service, or repo-specific
path rules were added.

## What changed

- **Docs topicality guardrail.** Docs-mode ranking now scores documentation files
  for topical overlap with task/path/artifact terms and aliases such as
  retrieval, search, ranking, rerank, troubleshooting, behavior, score, order,
  sort, and tie. Generic `index.md`/log/changelog pages receive a penalty in
  docs-shaped task lanes so topical docs outrank generic/unrelated specs.
- **Retrieval/ranking locality guardrail.** Refactor tasks infer generic locality
  anchors from strong direct file matches, then boost sibling/path-token-related
  files so adjacent retrieval/ranking source and tests stay close before broad
  runtime spillover.
- **Explainable score substrate.** `RankingScore` and `explain_task_file_score()`
  expose deterministic feature buckets for tests and audits: lexical, alias,
  role, document shape, topicality, symbol, freshness, language penalty,
  generic-index penalty, locality, artifact penalty, and total score.
- **Benchmark command alignment.** The wikiagent lane-regression prompt now runs
  `tests/retrieval_vs_analysis_benchmark.py` with pytest, matching the benchmark's
  pytest-style implementation.

## Regression coverage added

- Added deterministic tests in `tests/test_compile.py` to assert:
  - docs-mode topical docs outrank generic indexes and unrelated specs.
  - refactor-mode retrieval/ranking siblings stay ahead of runtime spillover.
  - ranking score explanations expose stable feature buckets.

## Validation

- `python -m unittest tests.test_compile`
- `python -m unittest`

## Client update guide

### pip

```bash
pip install -U ccw-mcp
```

### pipx

```bash
pipx upgrade ccw-mcp
```

### uvx launchers

```bash
uvx --refresh ccw-mcp==0.1.15 --help
```

### APM-managed MCP clients

```bash
apm install fmferrari/ccw --target copilot
```

Restart the editor or harness process after upgrading.

## Post-update verification

```bash
python -c "import ccw, inspect; print(ccw.__version__, inspect.getfile(ccw))"
```

Version must be `0.1.15`, and the path must not point to a vendored module copy.
