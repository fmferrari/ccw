# Release 0.1.16

## Summary

`ccw-mcp` `0.1.16` implements Phase 5H deterministic topical relevance scoring.
It corrects the remaining lane-quality audit failures without adding an LLM,
vector store, semantic-search service, or repo-specific path rules.

## What changed

- **Topicality-first lane preference.** Lane shape remains a preference, but weak
  docs/tests/source files no longer outrank strongly topical evidence merely
  because they match the requested lane.
- **Docs topicality gate.** Docs-mode ranking now lets topical source/tests beat
  unrelated docs/specs when no topical docs exist, while topical docs still lead
  when present.
- **Safer matching.** Exact tokens, morphology, phrase/co-occurrence, path terms,
  artifact title/search_text, and symbol names are primary. Synonym aliases remain
  capped and secondary.
- **Refactor specificity.** Refactor tasks preserve preferred source leadership
  over tests and cap frontend/cross-language code for non-frontend prompts.
- **Focused test specificity.** Test-shaped retrieval/ranking/tie prompts now
  favor focused test files over broad service/access-policy tests unless the broad
  files have stronger indexed topical evidence.

## Regression coverage added

- Added deterministic tests in `tests/test_compile.py` for:
  - topical source/tests beating unrelated docs in docs mode.
  - focused retrieval/ranking tests beating broad tests.
  - refactor source/locality behavior without runtime spillover.
- Updated the bugfix golden score after the shared topical scoring substrate
  increased the single-file score deterministically.

## Validation

- `python -m unittest tests.test_compile`
- `python -m unittest` → `185 tests` passed.
- Downstream wikiagent lane audit should be run with the pinned prompt after
  updating to `ccw-mcp==0.1.16`.

## Client update guide

### pip

```bash
pip install -U ccw-mcp
```

### pipx

```bash
pipx upgrade ccw-mcp
```

### uvx launchers

```bash
uvx --refresh ccw-mcp==0.1.16 --help
```

### APM-managed MCP clients

```bash
apm install fmferrari/ccw --target copilot
```

Restart the editor or harness process after upgrading.

## Post-update verification

```bash
python -c "import ccw, inspect; print(ccw.__version__, inspect.getfile(ccw))"
```

Version must be `0.1.16`, and the path must not point to a vendored module copy.
