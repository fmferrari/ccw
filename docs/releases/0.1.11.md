# Release 0.1.11

## Summary

`ccw-mcp` `0.1.11` addresses the remaining lane-quality audit findings: docs-mode
compilations now surface AGENTS.md and CONTEXT.md in the task evidence lane instead
of relegating them to the agentic context lane, and refactor-mode tasks penalize
test files more aggressively to ensure source files lead.

## What changed

- **Docs-mode task lane routing.** AGENTS.md and CONTEXT.md are no longer excluded
  from `_is_task_documentation_candidate`, so in docs mode they get the full
  documentation intent boost and appear in the task evidence lane alongside other
  wiki/spec files.
- **Stronger refactor test penalty.** The test-file score penalty for refactor-mode
  tasks increased from -7.0 to -12.0, making it harder for test files (even with
  strong lexical symbol matches) to outrank source files.

## Regression coverage added

- Updated 3 deterministic test assertions in `tests/test_compile.py` to reflect
  that AGENTS.md now routes to the task lane for docs-intent tasks.

## Validation

- `.venv/bin/python -m unittest` -> `177 tests` passed.
