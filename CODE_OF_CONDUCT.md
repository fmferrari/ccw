# Code of Conduct

CCW aims to be a respectful, technically rigorous, low-drama project.

## Expected behavior

- Be respectful and constructive in issues, pull requests, and review.
- Focus criticism on ideas, code, and behavior rather than people.
- Assume good intent, but be explicit when you disagree.
- Help keep discussions actionable and on topic.

## Unacceptable behavior

- Harassment, threats, discrimination, or personal attacks.
- Sexualized language or unwelcome sexual attention.
- Doxxing or publishing someone else's private information.
- Trolling, bad-faith disruption, or deliberately wasting contributor time.
- Spam or off-topic promotion.

## Moderation

Project maintainers may edit, remove, or reject comments, issues, pull
requests, or other contributions that violate this code of conduct. Repeated or
severe violations may lead to temporary or permanent bans from project spaces.

## Scope

This code of conduct applies to GitHub issues, pull requests, code review, and
other project-managed collaboration spaces.
